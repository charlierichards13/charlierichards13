## Featured Projects

### 🤖 StackTutor AI
AI-powered code review trainer designed to help computer science students learn how to debug instead of simply receiving answers.  
Built as a full-stack project with a mobile-first experience, authentication, and a polished learning workflow.

**Tech:** Expo/React Native, Supabase, TypeScript, Web, Auth, Full-Stack Development  
**Focus:** AI-assisted learning, code review, debugging practice, student-focused UX

---

### 🎯 C++ Ray Tracer
Collaborative ray tracing project focused on building a graphics engine from the ground up.  
Includes ray/sphere intersection logic, matrix transformations, computation testing, and lighting systems using ambient, diffuse, and specular shading.

**Tech:** C++, Makefile, Matrix Math, Ray Tracing, Phong Lighting  
**Focus:** Computer graphics, performance, math-heavy systems, collaborative development

---

### 🏀 AZ Road to Open
Full-stack tournament bracket platform for Arizona high school basketball.  
Built with **Next.js + Firebase**, featuring authentication, bracket locking, leaderboard logic, live updates, and admin tools.

**Tech:** Next.js, Firebase, JavaScript/TypeScript, Authentication, Realtime Data  
**Focus:** Full-stack development, sports tech, admin tooling, live tournament logic

---

### 🌵 HeatSafe Phoenix
Heat-safety web application designed to help Phoenix residents better understand weather-related risk.  
Built with a focus on **real-time safety guidance, public impact, and organization-focused heat planning tools**.

**Tech:** Web Development, Weather Data, Public Safety Tools  
**Focus:** Civic impact, real-world safety, data-driven guidance

---

### 🎮 DeathLine
A Unity action-roguelite prototype inspired by fast-paced combat and replayability.  
Built with **Unity + C#**, with focus on movement, combat feel, and gameplay systems.

**Tech:** Unity, C#, Game Systems  
**Focus:** Gameplay programming, combat feel, movement systems

---

### 🐾 Pet Image Classifier
Python-based image classification pipeline for pet image recognition.  
Focused on preprocessing, validation, and machine learning workflow design.

**Tech:** Python, Machine Learning, Image Classification  
**Focus:** Data preprocessing, validation, ML workflow design
